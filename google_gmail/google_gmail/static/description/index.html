Description

Google Gmail module native in version 14.0/15.0 adapted to work with version 11.0. Supports Google's new authentication method, as of May 30th, 2022.

Usage

Input your Gmail Oauth credentials (Client ID & Secret)

- Go to Settings
- Activate the developer mode
- Go to Parameters / System Parameters
- Fill in corresponding fields with client ID and secret

Details regarding Gmail-side configuration can be found in attached README file.


Configure Outgoing Server

- Go to Settings
- Activate the developer mode
- Go to Email / Outgoing Mail Servers
- Add new record
- Mark "Gmail" checkbox
- Input proper username
- Click "Get an Authorization Code"
- Accept conditions and copy / paste obtained code
- Click "Test Connection" to check if everything works
- Save record

Configure Incoming Server

- Go to Settings
- Activate the developer mode
- Go to Email / Incoming Mail Servers
- Add new record
- Mark "Gmail" checkbox
- Input proper username
- Click "Get an Authorization Code"
- Accept conditions and copy / paste obtained code
- Click "Test & Confirm" to check if everything works
- Save record

Installation

Download → Unzip → Move to your addons directory → Update the apps list → Find "Google Gmail" module → Install module

Details regarding Gmail-side configuration can be found in attached README file.
